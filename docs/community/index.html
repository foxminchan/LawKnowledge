Community Support

Need help?​

This project is maintained by a dedicated group of people.

If you need help with the Law Knowledge Base, you can reach out to the community, or the team via:

  • GitHub Discussions
  • Telegram.

Donate​

If you like this project, please consider donating to support it. Support will allow the maintainers to dedicate more time for maintenance and new features for everyone.

Thank you to everyone who will or has donated money to the collective and therefore has supported the project and everyone who contributed to it. It really means a lot ❤️

Technical steering committee​

The technical steering committee (TSC) is responsible for the overall technical direction of the Law Knowledge Base project. The TSC is made up of the project maintainers.
